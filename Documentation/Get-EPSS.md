# Get-EPSS

## SYNOPSIS
Get EPSS information

## SYNTAX

```
Get-EPSS [[-CVE] <String[]>] [<CommonParameters>]
```

## DESCRIPTION
Get EPSS information or score for specific CVE

## EXAMPLES

### EXAMPLE 1
```
Get-EPSS -CVE 'CVE-2022-27225'
Get the EPSS score for CVE 'CVE-2022-27225'
```
